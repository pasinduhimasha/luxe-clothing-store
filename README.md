# Smart Hostel Management System

A full-stack **Smart Hostel Management System** built using **React (Vite)** and **Firebase**, designed to manage hostel operations such as rooms, tenants, payments, notices, complaints, and admins in one centralized platform. Suitable for **academic, portfolio, or real-world use**.

---

## Live Demo

[View Smart Hostel Management System Live](https://smart-hostel-management-system-lvjn-59gja4zf6.vercel.app/)

---

## Features

- User & Admin Authentication
- Room Management (add/edit/delete)
- Tenant Management with real-time updates
- Payment tracking (pending/paid)
- Notice management
- Complaints & inquiries handling
- Admin management & secure roles

---

## Tech Stack

- **Frontend:** React (Vite), React Router DOM, Tailwind CSS  
- **Backend / Services:** Firebase Authentication & Firestore  
- **Deployment:** Vercel
